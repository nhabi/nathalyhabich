Art Direction & Production Design

The project is fully designed, structured and ready for execution.

This service transforms an idea into a complete visual system with clear direction, spatial design and a real foundation for production.

Start Your Project

What it is and what it helps build

A full structural design service created to develop the visual world of a project with coherence, clarity and direction.

This is not only about defining an aesthetic. It is about building a complete visual proposal that organizes how the project looks, how it holds together and how it should be executed.

It turns an idea into a complete, clear and executable visual system so the project can move forward with structure instead of improvisation.

What it solves

  • Helps define the project visually with a clear direction
  • Builds a coherent visual language across concept, space and narrative
  • Organizes decisions before purchasing, installation or execution begins
  • Translates creative intent into a system production can actually use
  • Reduces errors, contradictions and unresolved gaps before the next stage

Included phases

Phase 0

Strategic project reading

Initial reading of objectives, context, constraints and the level of visual development required.

Phase 1

Foundational visual direction

Definition of the visual axis that organizes decisions and sets criteria for the full design process.

Phase 2

Concept development

Development of visual narrative, structured references and key atmosphere decisions.

Phase 3

Spatial / set design application

Translation of the concept into space: layout, composition, circulation and functional logic.

Phase 4

Visual system and materiality

Consolidation of color, materials, props and visual unity criteria across elements.

Phase 5

Preparation for execution

Delivery of the technical and visual package so production can execute with operational clarity.

Deliverables

  • Structured PDF
  • Technical plans
  • Shopping list

Reference view of the delivery structure for design, layout and technical documentation.

What’s not included

  • Purchasing
  • Installation
  • Filming
  • On-site execution supervision
  • Direct team management during production

This service leaves the project fully designed and prepared for execution, but it does not include on-site direction during production. If the project also requires direction during installation, purchasing or filming, the appropriate next step is Art Direction in Production.

Limits and revisions

Up to 3 main sets

2 revision rounds per phase

Changes outside an approved phase may require an adjustment in scope, timing or cost.

Result

A visually designed, structured and production-ready project with greater clarity, coherence and operational order.

This service is right for you if...

Good fit if

  • the project already needs full visual development
  • a general idea is no longer enough and it needs to be turned into a system
  • you need structure before moving into execution
  • you want design, narrative and production to work in alignment
  • you do not yet need on-site direction, but you do need the project to be fully resolved before that stage

Not the right fit if

  • the project is still too early and what you need first is strategic clarity
  • the project is already in production and requires active direction during purchasing, installation or filming
  • you are only looking for execution without a serious design phase

What sets this apart from production-phase support

This is where the project gets designed and structured. It does not include directing execution on site.

That means you receive a complete proposal that is ready to move into production, but supervision of purchasing, installation, prelight and visual continuity belongs to the next level of service.

When the project no longer needs just ideas, but a real visual structure, this is the right service.

If you need to ground the concept, build the complete visual system and leave the project ready for execution, this is where that foundation gets built.
