Servicio 03

Dirección de Arte en Producción

El proyecto no solo se diseña. Se dirige en ejecución.

Este servicio integra diseño, dirección visual y acompañamiento en producción para asegurar que el mundo planteado se sostenga con coherencia hasta el montaje, el prelight y el rodaje.

Iniciar proyecto

Qué es y qué asegura en ejecución

Un servicio integral para proyectos que necesitan dirección de arte durante la ejecución, no solo en la etapa de diseño.

Aquí la dirección visual no se queda en un documento. Se sostiene en decisiones reales de producción, compras, montaje, ajustes en campo y validación final.

Asegura que el proyecto mantenga coherencia visual desde la definición hasta la ejecución, evitando que el diseño se diluya al pasar a producción.

Qué resuelve

  • sostener el criterio visual durante compras, montaje y rodaje
  • tomar decisiones de arte en tiempo real sin perder coherencia
  • alinear diseño, producción y ejecución bajo una misma dirección
  • corregir desvíos antes de que afecten el resultado final
  • evitar que el proyecto termine ejecutado por inercia, sin control visual real

Fases incluidas

Fase 0

Lectura estratégica del proyecto

Se alinea el objetivo visual con el alcance real de producción, tiempos y prioridades de ejecución.

Fase 1

Dirección visual base

Se define el criterio rector que guiará decisiones de arte, materialidad y puesta en escena durante todo el proceso.

Fase 2

Desarrollo conceptual

Se traduce la narrativa del proyecto en lenguaje visual aplicable para cada frente operativo.

Fase 3

Diseño aplicado al espacio / set

Se ajusta el diseño a condiciones físicas, técnicas y de uso real para evitar brechas en montaje y rodaje.

Fase 4

Sistema visual y materialidad

Se definen paletas, acabados, texturas y especificaciones para sostener coherencia en proveedores y compras.

Fase 5

Preparación para ejecución

Se ordenan decisiones previas al montaje para que el equipo pueda ejecutar con claridad y sin improvisación.

Fase 6

Supervisión de compras

Se valida que materiales y recursos adquiridos mantengan el estándar visual definido.

Fase 7

Dirección de montaje

Se dirige en campo la implementación del diseño y se corrigen desvíos en tiempo real.

Fase 8

Prelight y validación visual

Se revisa composición, lectura y continuidad antes de iniciar rodaje o apertura.

Fase 9

Continuidad y control final en ejecución

Se asegura que la ejecución final conserve coherencia con lo planteado en todo el recorrido de producción.

Estructura económica

Este servicio se divide en tres partes:

Honorarios de dirección

Mi trabajo de dirección de arte y toma de decisiones durante el proceso.

Honorarios del equipo

Asistente de arte, supervisor o apoyo operativo, según el alcance del proyecto.

Presupuesto de producción

Compras, materiales, proveedores, alquileres y ejecución, pagados directamente por el cliente.

No se trabaja por porcentaje.

Qué no incluye

  • Presupuesto de producción
  • Pago a proveedores
  • Honorarios del equipo adicional
  • Recursos de montaje no contemplados por cliente
  • Cambios de alcance fuera de lo aprobado

Este servicio dirige la ejecución, pero no absorbe costos de producción ni reemplaza la estructura operativa que el proyecto necesita para realizarse bien.

Requiere

Asistente de arte o apoyo operativo

Equipo mínimo de montaje según el proyecto

Presupuesto de producción definido o viable

Cronograma aprobado antes de ejecución

Este servicio no funciona bien si se espera dirección de arte completa sin equipo, sin estructura o sin capacidad real de ejecución.

Límites y revisiones

Alcance definido según presupuesto y complejidad del proyecto

Cambios fuera de fase afectan tiempo y costo

Ajustes después de aprobación requieren reevaluación

El servicio se estructura en función del cronograma real de producción

Resultado

Un proyecto diseñado y ejecutado bajo una dirección visual coherente, con control real sobre las decisiones que afectan el resultado final.

Este servicio encaja si...

Sí es para ti si

  • el proyecto ya pasó o está por pasar a ejecución
  • no basta con tener diseño: hace falta sostenerlo en producción
  • necesitas dirección de arte real durante compras, montaje y rodaje
  • quieres evitar que el resultado final pierda fuerza frente a lo planteado

No es para ti si

  • todavía necesitas solo claridad estratégica
  • aún no estás en etapa de producción
  • buscas únicamente una propuesta visual, sin acompañamiento en ejecución
  • no existe equipo mínimo ni condiciones reales para sostener el proceso

Qué diferencia este servicio de Dirección de Arte y Diseño de Producción

En el servicio anterior el proyecto queda diseñado y preparado para ejecutarse. Aquí, además de diseñarse, se dirige la ejecución.

Eso implica acompañar decisiones reales de producción, sostener el criterio visual en campo y validar que lo construido conserve coherencia con lo planteado.

Cuando el proyecto necesita dirección visual real durante la ejecución, este es el nivel correcto.

Si no basta con diseñar y hace falta dirigir compras, montaje, equipo y continuidad visual, este servicio es el que sostiene el proyecto hasta su materialización.
